
PROJECT TITLE: Parking Spot System with A Graphical User Interface (GUI)

PURPOSE OF PROJECT: A small system that help manage cars at the parking site for a company.

VERSION or DATE: Version 1

HOW TO START THIS PROJECT: Run Application.java

AUTHORS: Xiaodan Cao

USER INSTRUCTIONS: 

User first establishes a car park by providing the number of slots for staff and visitors and 
then will able to find a slot, add a slot, delete a slot, park a car and provide a list of all slots included in the car park.
